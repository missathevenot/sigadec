# SIGADEC
**Système Intégré de Gestion Administrative et Décisionnelle du Cadastre**
Direction Générale des Impôts — Direction du Cadastre, Côte d'Ivoire

## Démarrage rapide
```
npm install && npm run dev
```

## Build production
```
npm run build
```

## Déploiement
Push sur `main` → GitHub Actions → GitHub Pages automatiquement

## Accès de démonstration
| Email | Rôle |
|---|---|
| directeur@cadastre.dgi.ci | Directeur du Cadastre |
| admin@cadastre.dgi.ci | Administrateur |
| chef.scoa@cadastre.dgi.ci | Chef de Service (SCOA) |
| secretariat@cadastre.dgi.ci | Secrétariat |

## Modules
- Tableau de bord — vue consolidée par rôle
- Diligences — suivi et mise à jour des tâches
- Informations / Divers — notes et circulaires
- Documentation — dépôt et archive des rapports
- Charte Éthique — planning et soumission mensuelle
- Planning Annuel — calendrier Charte + Comptes-Rendus
- Courriers — courriers émis et reçus
- Émissions / Recettes — données fiscales (accès restreint)
- Administration — validation des comptes
